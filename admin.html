🔥 Firebase Rules — ছবি save না হলে এটা করুন

Firebase Console → Firestore Database → Rules তে নিচের rules দিন:

rules_version = '2';
service cloud.firestore {
  match /databases/{database}/documents {
    match /{document=**} {
      allow read: if true;
      allow write: if request.auth != null;
    }
  }
}

Firebase Console → Storage → Rules তে নিচের rules দিন:

rules_version = '2';
service firebase.storage {
  match /b/{bucket}/o {
    match /{allPaths=**} {
      allow read: if true;
      allow write: if request.auth != null;
    }
  }
}
